---
title: Can't Edit Project Documentation
description: Resolve editing issues in project documentation.
---

## Symptoms

- Edit button is missing
- Changes are not saving

## Possible Causes

- Insufficient permissions
- Session timeout

## Quick Fix

<Steps>
  <Step title="Check your login">
    ````
    Make sure you are logged in with the correct account.
    ````
  </Step>
  <Step title="Request access">
    ````
    Contact the project admin to grant you edit permissions.
    ````
  </Step>
</Steps>
