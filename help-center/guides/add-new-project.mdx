---
title: Adding a New Project
description: Create a new project documentation area.
---

## Steps

<Steps>
  <Step title="Navigate to Dashboard">
    ````
    Go to https://alexanderanokhin.com/dashboard
    ````
  </Step>
  <Step title="Click Add Project">
    ````
    Select the "Add Project" button and enter project details (name, description).
    ````
  </Step>
  <Step title="Save and Start Editing">
    ````
    Click "Save". Your new project area is ready for documentation.
    ````
  </Step>
</Steps>
