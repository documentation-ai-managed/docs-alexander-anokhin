## Overview

Welcome to Alexander Anokhin's documentation space. You use this platform to organize, manage, and share all your project documentation efficiently. It serves as your central hub for creating structured docs, collaborating with teams, and maintaining version control. Start by exploring the intuitive interface designed for developers, writers, and project managers.

<Callout kind="info">
This space supports MDX for rich, interactive documentation with components like tabs, steps, and cards.
</Callout>

## Quick Navigation Guide

Get started navigating your documentation space in minutes. Follow these steps to set up and explore.

<Steps>
  <Step title="Access Your Space" icon="login">
    Log in at `https://docs.example.com` using your credentials.
  </Step>
  <Step title="Create a Page" icon="plus">
    Click the `+ New Page` button in the sidebar. Choose a title and start editing.
  </Step>
  <Step title="Organize Content" icon="layers">
    Drag pages into folders or use the search bar to find content quickly.
  </Step>
  <Step title="Publish Changes" icon="upload">
    Preview your docs, then hit `Publish` to make them live.
  </Step>
</Steps>

## Example MDX Usage

Enhance your pages with components. Here's a multi-language code example for embedding a simple script.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Fetch documentation data
async function loadDocs() {
  const response = await fetch('/api/docs');
  return response.json();
}
loadDocs();
```

```python
# Fetch documentation data
import requests

response = requests.get('https://api.example.com/docs')
docs = response.json()
print(docs)
```
</CodeGroup>
